The Speaker ROM
=======================

Originally published by Superior Software as SPEECH 1986. 
Republished by Xavier Grossetete as a BASIC listing in CPC Infos 1991. 
Retyped by NoRecess 2011. 
Amstrad ROM version by Chris Perver 2026.

Compilation instructions
------------------------

You will need the following tools, placed in the compile folder:- 
  * RASM Z80 assembler for Linux

CD to the 'compile' folder.
Run the appropriate bash script to compile the ROM.
The files will be placed in the 'build' folder.
