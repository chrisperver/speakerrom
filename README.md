**THE SPEAKER**

Published by Superior Software as SPEECH 1986.
Republished by Xavier Grossetete in CPC Infos 1991.
Retyped by NoRecess 2011.
ROM version by Chris Perver 2026.

To compile, CD to the 'compile' folder and run the makerom.bash script.
You will need to have RASM Z80 assembler installed.
The compiled files will be placed in the 'build' folder.
